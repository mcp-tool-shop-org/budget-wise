# Code of Conduct

We want NextLedger to be a welcoming project.

## Our standards

- Be respectful and constructive.
- Assume good intent; ask clarifying questions.
- Keep feedback specific and actionable.
- No harassment or hateful conduct.

## Scope

This applies to issues, pull requests, discussions, and other project spaces.

## Enforcement

If behavior is not aligned with this code of conduct, maintainers may remove content or restrict participation.

## Reporting

If you need to report a conduct issue, open a private report via the repository maintainers.
