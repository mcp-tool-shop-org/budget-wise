# Phase 12 Evidence Folder

Store Phase 12 evidence here.

## Layout

- `docs/phase12/screenshots/commit-01/`
- `docs/phase12/screenshots/commit-02/`
- ...
- `docs/phase12/screenshots/commit-10/`

Each folder should contain:

- Before/after screenshots (`.png` recommended)
- Optional short recordings (`.gif` or `.mp4`) for soak commits
- If needed: a brief `NOTES.md` explaining what the evidence shows

See `docs/PHASE12_AUDIT.md` for the required audit log per commit.
